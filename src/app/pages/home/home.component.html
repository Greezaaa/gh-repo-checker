Welcome to GitHub Issues API reader

Please enter repository link into the searchbox to get issues info

How it's work?

It's simple. Just enter repository link into the search box and hit "enter" to get an info.

The app will make a request to the GitHub API and return repository and list of this repository issues.

Also you can click on the issue to check more details directly from GitHub.

Enjoy!

How I made it?
  • I used Angular 15, TailwindCSS, Angular Router, NgRX, TypeScript HTML5, GitHub API.
  • Angular 15 - for frontend
  • TailwindCSS - for styling
  • Angular Router - for routing
  • NgRX - for state management
  • TypeScript - for type checking
  • HTML5 - for markup

Why TailwindCSS and not MAterialUI or Bootstrap
